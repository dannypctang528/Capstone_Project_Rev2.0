Capstone Project: Stack Overflow Developer Survey Analysis (2024)

This repository contains my capstone project analyzing the 2024 Stack Overflow Developer Survey.  
The project explores developer preferences, compensation trends, and satisfaction across technologies, platforms, and demographics using Python, Looker Studio, and Google Slides.
---

Contents
- **`Capstone_v2_2_DannyTang.ipynb`** → ETL pipeline, feature engineering, and exploratory data analysis in Python  
- **`presentation.pdf`** → Final Google Slides presentation summarizing insights and visual storytelling  
- **`dashboard.pdf`** → Exported Looker Studio dashboards with interactive charts and filters  
---

Key Highlights
- Cleaned and transformed survey data (dropna, deduplication, IQR outlier removal)
- Feature engineering (age ranges → midpoints, simplified education levels)
- Interactive dashboards (bubble charts, scatter plots, sunburst, treemaps)
- Insights on programming languages, platforms, databases, compensation, and satisfaction
- Executive summary with clear observations and professional presentation
---

Tools & Technologies
- **Python** (pandas, numpy, matplotlib)
- **Google Looker Studio**
- **Google Slides**
- **MS Excel & VBA**
- **Stack Overflow Developer Survey 2024 dataset**
---

Insights
- Top programming languages remain consistent, with TypeScript gaining ground
- SQL-based databases dominate, though NoSQL options show niche adoption
- Compensation correlates with satisfaction, but cultural and non-monetary factors also play a role
- Younger, well-educated respondents form the majority of the survey base
